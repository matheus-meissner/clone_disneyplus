# 🎬 Clone Disney Plus 🎬

Bem-vindo ao **Clone Disney Plus**! Este projeto recria a interface de usuário da plataforma de streaming Disney Plus, proporcionando uma experiência visual semelhante à plataforma oficial. 🎥✨
